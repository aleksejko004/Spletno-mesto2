Osnovni SQL stavki

Začnimo z osnovnim SQL stavkom, ki se uporablja za pridobivanje podatkov iz podatkovne baze. Poglejmo si primer in razložimo, kaj pomeni.

Predstavljajte si, da imate tabelo z imenom customers, ki vsebuje informacije o vaših strankah. Če želite videti vse podatke iz te tabele, lahko uporabite naslednji SQL stavek:

SELECT * FROM customers;

Razlaga: Ta stavek je sestavljen iz dveh ključnih delov:

  • SELECT *: To pomeni, da želite izbrati vse stolpce iz tabele. Zvezdica (*) je kratica za "vse".
  • FROM customers: To določa, iz katere tabele želite pridobiti podatke.

Torej, ta preprosti SQL stavek vam bo vrnil vse vrstice in vse stolpce iz tabele customers.

Dodajanje novih podatkov (INSERT)

Za dodajanje novih zapisov v tabelo uporabljamo stavek INSERT INTO. Poglejmo si primer, kako dodati novo stranko v tabelo customers.

INSERT INTO customers (customer_name, email, city) VALUES ('Ana Novak', 'ana.novak@email.si', 'Ljubljana');

Kaj ta stavek naredi? Ta stavek dodaja novo vrstico v tabelo customers. Določeni so stolpci (customer_name, email, city) in vrednosti, ki se bodo vstavile v te stolpce ('Ana Novak', 'ana.novak@email.si', 'Ljubljana'). Vrstni red vrednosti mora ustrezati vrstnemu redu stolpcev.

Spreminjanje obstoječih podatkov (UPDATE)

Za spreminjanje obstoječih zapisov v tabeli uporabljamo stavek UPDATE. Na primer, če želimo posodobiti e-poštni naslov stranke z imenom 'Ana Novak'.

UPDATE customers SET email = 'ana.n@novi-email.si' WHERE customer_name = 'Ana Novak';

Kaj ta stavek naredi? Ta stavek posodablja vrstice v tabeli customers. Ključna dela sta:

  • SET email = 'ana.n@novi-email.si': Določa, kateri stolpec naj se posodobi in na katero vrednost.
  • WHERE customer_name = 'Ana Novak': Določa, katere vrstice naj se posodobijo. Pomembno je uporabiti klavzulo WHERE, sicer se bodo posodobili vsi zapisi v tabeli!

Brisanje podatkov (DELETE)

Opozorilo pri brisanju

Če izpustite klavzulo WHERE pri stavku DELETE, se bodo izbrisali vsi zapisi iz tabele! Bodite zelo previdni pri uporabi DELETE stavka.

Za brisanje obstoječih zapisov iz tabele uporabljamo stavek DELETE FROM. Na primer, če želimo izbrisati stranko z imenom 'Ana Novak'.

DELETE FROM customers WHERE customer_name = 'Ana Novak';

Kaj ta stavek naredi? Ta stavek briše vrstice iz tabele customers. Podobno kot pri UPDATE, klavzula WHERE (WHERE customer_name = 'Ana Novak') določa, katere vrstice naj se izbrišejo.

Ustvarjanje in spreminjanje struktur

Ustvarjanje nove tabele (CREATE TABLE)

CREATE TABLE orders ( order_id INT PRIMARY KEY AUTO_INCREMENT, customer_id INT, order_date DATE, total_amount DECIMAL(10, 2) );

Ta stavek ustvari novo tabelo z imenom orders s štirimi stolpci, vključno s primarnim ključem.

Spreminjanje strukture (ALTER TABLE)

ALTER TABLE orders ADD delivery_address VARCHAR(255);

Ta stavek spreminja tabelo orders tako, da ji doda nov stolpec z imenom delivery_address.
